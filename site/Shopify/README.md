
# 如何自建Shopify?

## 建站前提条件
#### 1.顶级域名
##### 2.收款方账号
* 例如PayPal企业账号(企业资质)
* 或shopify payments
* 或Stripe信用卡收款
#### 3.支持外币信用卡
#### 4.网站主题
* shipify免费主题(功能有限)
* 主题购买:
	* [Themeforest](https://link.zhihu.com/?target=https%3A//www.qingsongb2c.com/goto/themforest-shopify-themes/)

#### 5.产品, 货源
